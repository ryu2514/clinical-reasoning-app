# 臨床推論統合アプリ

理学療法士向けの臨床推論トレーニングアプリケーション

## 機能

### 📝 基本情報管理
- 患者の基本情報を一元管理
- 全タブで共通の患者IDシステム

### 🔍 評価タブ
- セッション管理機能
- 詳細な評価項目の記録
- CSV出力・読み込み機能

### 📋 症例分析タブ
- 三角ロジック（データ・根拠・結論）
- AIによるフローチャート自動生成
- 治療選択の構造化入力
- CSV/PNG出力機能

### 💊 治療介入タブ
- 運動療法・徒手療法・物理療法の計画
- 効果判定（即時・短期・長期）
- 振り返りと学びの記録
- CSV出力・読み込み機能

## データ管理

### エクスポート機能
- **評価データ**: CSV形式
- **症例分析データ**: CSV形式
- **治療介入データ**: CSV形式
- **フローチャート**: PNG画像

### インポート機能
- 講師が修正したCSVファイルを読み込んで反映可能
- 週次ZOOM面談でのフィードバック循環に最適

## 使用技術

- React 18
- Babel (ブラウザ内トランスパイル)
- Gemini 2.5 Pro API (フローチャート生成)
- html2canvas (画像出力)
- localStorage (データ永続化)
